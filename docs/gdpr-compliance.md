# GDPR Compliance Documentation
## Family Advent Calendar Application

### Data Controller Information
- **Controller**: Family Advent Calendar LLC
- **Contact**: privacy@familyadventcalendar.com
- **Data Protection Officer**: privacy@familyadventcalendar.com

### Legal Basis for Processing
- **Consent**: User consent for data processing
- **Contract**: Necessary for service provision
- **Legitimate Interest**: Product improvement and security
- **Legal Obligation**: Compliance with applicable laws

### Data Subject Rights

#### Right to Access
- Users can request a copy of all their personal data
- Data export available through `/api/auth/export` endpoint
- Export includes all personal information, calendar data, and usage history

#### Right to Rectification
- Users can update their information through account settings
- Corrections processed within 30 days

#### Right to Erasure ("Right to be Forgotten")
- Complete data deletion available through account deletion
- All personal data permanently removed within 30 days
- Some anonymized data may be retained for legal/compliance reasons

#### Right to Data Portability
- Data export in machine-readable format
- Compatible with other calendar/family applications

#### Right to Restriction
- Processing can be restricted upon user request
- Data marked as restricted and not used for processing

#### Right to Object
- Users can object to data processing for direct marketing
- Processing stopped immediately upon objection

### Data Processing Activities

#### Personal Data Collected
- **Identity Data**: Name, email address, date of birth
- **Profile Data**: Child information, interests, preferences
- **Usage Data**: Application interactions, analytics
- **Technical Data**: IP address, device information, browser data

#### Data Recipients
- **Hosting Provider**: Supabase (EU-based)
- **Analytics**: Vercel Analytics (anonymized)
- **AI Services**: OpenAI (data processed per their privacy policy)
- **Push Notifications**: Firebase (Google)

### International Data Transfers
- All data stored in EU-based infrastructure
- No transfers to third countries without adequate protection
- Standard Contractual Clauses used where applicable

### Data Security Measures
- **Encryption**: Data encrypted at rest and in transit
- **Access Controls**: Role-based access with principle of least privilege
- **Audit Logging**: All data access logged and monitored
- **Regular Security Assessments**: Penetration testing and vulnerability scans

### Data Breach Notification
- Breaches affecting personal data reported within 72 hours
- Affected users notified within additional time as needed
- Regulatory authorities notified as required

### Cookies and Tracking
- Essential cookies only for application functionality
- No third-party tracking or advertising cookies
- Analytics data anonymized and aggregated

### Children's Data (COPPA Compliance)
- Parental consent required for children under 13
- Separate processing rules for children's data
- Enhanced security measures for child accounts
- Age verification and parental controls implemented

### Contact for GDPR Inquiries
- **Email**: privacy@familyadventcalendar.com
- **Response Time**: Within 30 days
- **Complaints**: Users can lodge complaints with supervisory authorities

### Data Protection Impact Assessment
- DPIA conducted for high-risk processing activities
- Regular reviews and updates to DPIA
- Mitigation measures implemented
